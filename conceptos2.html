Tres principales valores


Inline

El elemento genera uno o más cuadros de elemento en línea.

Ejemplo

los elementos li, por defecto se muestran uno en cada linea, con el valor de display inline quedaran en una sola linea como se observa en la imagen.

Figura 1.inline. Elaboración propia.

Block

El elemento genera un cuadro de elemento de bloque.

Ejemplo

Si los elementos a, que por defecto se muestran en linea,queremos cambiarlos con el valor de display Block quedaran en lista.

Figura 2.block. Elaboración propia.
Inline-Block

El elemento genera una caja block pero que se comporta como si fuera en linea admitiendo otros elementos en la misma línea.

Ejemplo

Los siguientes cuadros por defecto ocupan en 100% del ancho, para colocarlos en linea necesitamos el valor display inline-block
